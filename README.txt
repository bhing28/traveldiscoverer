TravelDiscoverer static website package

Upload these files to GitHub Pages or Cloudflare Pages.

Included:
- index.html homepage
- flights.html with Manila MNL destination code support
- hotels.html
- contact.html with FormSubmit placeholder
- destination pages for Manila, Cebu, Tokyo, Seoul
- service pages for insurance, visa, transfers, flight compensation
- sitemap.xml and robots.txt

Before launch:
1. Replace info@traveldiscoverer.com in contact.html with your verified FormSubmit email or form provider.
2. Replace placeholder affiliate buttons with approved TravelPayouts, VisaHQ, Kiwitaxi, Insubuy, Compensair, and travel insurance links.
3. Upload to GitHub, then connect the repository to Cloudflare Pages or GitHub Pages.
